# Sentinel Core Frontend

React frontend for the Sentinel Core Security Platform.

## Stack

- React 19
- Vite 8
- React Router
- Axios
- Tailwind CSS 4
- Lucide React
- Oxlint

## Run Locally

```powershell
npm install
npm run dev
```

If PowerShell blocks `npm.ps1`, use:

```powershell
npm.cmd run dev
```

The app runs at:

```text
http://localhost:5173
```

The backend API is expected at:

```text
http://localhost:8080
```

## Scripts

```powershell
npm.cmd run dev
npm.cmd run build
npm.cmd run lint
npm.cmd run preview
```
